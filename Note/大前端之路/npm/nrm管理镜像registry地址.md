## 安装nrm

```shell
npm install -g nrm
```

## 查看镜像列表

```shell
nrm ls
```

![查看镜像列表](https://zxaq.oss-cn-shenzhen.aliyuncs.com/loongfile/youdao/WX20200907-102103%402x.png)

## 切换镜像

```shell
nrm use taobao
```

> ps：r_name镜像名字， r_url镜像地址

## 在nrm添加自己的镜像地址

```shell
nrm add r_name r_url
```

## 删除

```shell
nrm del r_name
```

## 测试镜像的相应速度

```shell
nrm test r_name
```
